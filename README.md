# 🌱 HabitFlow

> Build better habits. Backed by science. Powered by community.

HabitFlow is an open-source social habit tracking platform where people build science-backed habits, track their progress, and grow together through shared knowledge and real experience.

---

## Why HabitFlow?

Most habit trackers are just checklists. HabitFlow is different:

- **Science-backed templates** — every habit comes with links to research. Not "read more", but *why 25 minutes, why 3 times a week, why this rest interval*.
- **Community-verified** — users share their real experience in topic forums. What works, what doesn't, adjusted for real life.
- **Crowdsourced insights** — anonymized aggregate data from thousands of users creates a living knowledge base. The more people use it, the smarter it gets.
- **Your habits, your way** — pick templates or build your own. Track streaks, notes, and progress at your own pace.

---

## Core Features

| Feature | Description |
|---|---|
| 📋 Habit Templates | Curated habits per category (fitness, reading, sleep, nutrition...) |
| 🔗 Source Links | Every template links to the research behind it |
| 📈 Progress Tracking | Daily logs, streaks, personal notes |
| 💬 Topic Forums | Discuss habits by category — like Threads, but focused |
| 👥 Social Profiles | Share progress, follow others, build accountability |
| 📊 Community Data | Aggregated insights from real user experience |

---

## Vision

> *"What if habit science wasn't locked in academic papers, but lived inside the tool you use every day?"*

Phase 1 — Personal tracker (current)  
Phase 2 — Social layer: profiles, forums, following  
Phase 3 — Community research: surveys, aggregated habit data, partner product testing with community approval  

The long-term goal: a platform where people don't just track habits — they collectively discover what actually works.

## Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/habitflow.git
cd habitflow

# Backend
cd backend
./mvnw spring-boot:run

# Frontend
cd frontend
npm install
ng serve
```
